銷售訂單查詢 — 操作說明書

查 ERP 銷售訂單明細 → 一鍵複製 → 貼進「出單表」，省去逐欄手打。
👤 對象：全體同仁 📍 位置：美編天地 → 銷售訂單查詢

一、整體流程（先看這個）

  • 1（必要時）按「🔄 同步銷售訂單」抓 ERP 最新資料；平常查得到就不用每次按。
  • 2輸入訂單號碼 → 按「查詢」完整或部分單號都可以，按 Enter 也行。
  • 3勾選要的明細列可逐列勾，或點表頭勾選框全選。
  • 4按「複製 N 筆」N＝你勾的筆數。
  • 5到出單表，點 A 欄該列貼上（Ctrl+V）欄位會自動對位。
  • 6補上 R 欄「盤數」等要人工填的欄位貼上不會覆蓋這些欄。

二、同步銷售訂單

  • 右上角 🔄 同步銷售訂單：把 ERP 最新的銷售訂單抓進系統。
  • 什麼時候按：查不到剛開的單、或資料看起來是舊的，就先按一次同步。
  • 會跳出進度視窗，通常 30～120 秒，請勿關閉視窗；完成後可按「重新載入查詢」。
放心：同步只是用 ERP 最新資料覆寫本地畫面，不會改到 ERP。

三、查詢（可輸入部分單號）

在「訂單號碼」欄輸入後按 查詢（或直接按 Enter）。

  • 完整單號：例如 SO260603001 → 列出該單所有明細。
  • 部分單號（新功能）：例如只打 0603 → 列出所有單號含 0603 的訂單，不分大小寫。適合「只記得一段」或「想一次撈某批」。
筆數上限：一次最多顯示 500 筆。若底部出現「已達 500 筆顯示上限，請輸入更完整的單號」，代表撈太多，多打幾個字縮小範圍即可。

查不到時：① 確認單號沒打錯　② 先按一次「同步銷售訂單」再查。

五、複製到出單表（重點）

  • 勾選：點每列最左的勾選框；或點表頭勾選框全選／全不選（也可直接點整列切換）。
  • 按 複製 N 筆，出現「已複製 N 筆！」即成功。
  • 到出單表，點 A 欄（工單編號）要貼的那一列，按 Ctrl+V。

六、常見問題

「出貨備註」欄常常是「—」空白？
出貨備註抓的是 ERP 的「備註2」。SO 類訂單的備註2 多半沒填，內容通常集中在 PACKING 欄。這是 ERP 來源資料本身就空，不是系統漏抓。若你的出貨指示其實打在別的欄位，請回報系統管理員調整。
貼到出單表後欄位錯一格？
新版已修正對位。若仍錯位，確認你是貼在 A 欄那一列（不是從別欄開始貼），且出單表欄位順序沒被改動。
交期顯示紅色是什麼意思？
紅色＝交期已過（逾期）；綠色＝未到期。方便一眼抓出趕件。
複製後貼到 Excel 變成一格塞全部？
請用 Ctrl+V 直接貼（系統已用 Tab 分隔分欄）；若用「選擇性貼上 → 純文字」可能不分欄。
查詢結果的排序？
先依訂單號、再依序號由小到大排列。
