回報異常 → 填處理內容 → 設定缺失處置 → 一鍵印出「品質異常處理單」Excel。
👤 對象：品保、生產、美編等相關同仁 📍 位置：首頁 → 異常單回報 ／ 品保專區 🗓️ 版本：2026/07/24

一、整體流程（先看這個）

  • 1發現異常 → 填「異常回報單」首頁「異常單回報」→ 建立異常單。填 MOT 製令單號，訂單資料自動帶出。
  • 2責任單位填「異常單處理」首頁「異常單回報」→ 異常單處理。填處理紀錄與即時處理方式。
  • 3品保在「異常紀錄表」補齊資料指定缺失人員、填原因分析／預防及修正方式、異常數量。
  • 4到「異常人員缺失單處理作業」設定缺失處置選口頭矯正／口頭警告／小過／大過。
  • 5勾選要印的事件 → 按列印下載 Excel，一位缺失人員一張單，開啟後直接列印簽核。

二、填異常回報單（新版重點）

位置：首頁 異常單回報 → 建立異常單。

相關單號建議填「MOT 製令單號」

  • 在「相關單號」欄輸入 MOT26071400701 這類製令單號，停一下系統會自動查詢。
  • 查到後會出現綠色提示，並自動帶出：訂單編號（SO）、品項編碼、產品名稱 —— 這三項都不用自己填。
  • 右邊「本單數量」會顯示該製令的數量，方便對照填「異常數量」。
  • 沒打行號也可以（例：MOT260714007），系統會帶第一行。
也可以填別的單號：填 SO、RO 或其他單號一樣可以送出，只是不會自動帶資料。出現黃字「查無此製令單號」時，代表系統查不到，會照你輸入的內容存檔，不會擋住你送出。

異常數量怎麼填

  • 「異常數量」＝這次缺失造成的不良／損失數量。
  • 填了以後，缺失單列印時會自動帶成「不良數」，並用異常數量 ÷ 訂單量自動算出「不良率」。
  • 不確定就先留空，之後可在「異常紀錄表」編輯時補填。

三、異常紀錄表（查詢與補資料）

位置：品保專區 → 異常紀錄表。

日期區間查詢（新功能）

  • 篩選區有起始日期／結束日期兩個欄位，點一下會跳出小日曆。
  • 下面五顆快選鈕：本月／上個月／近三個月／近半年／近一年，點了自動填好日期。
  • 可搭配部門、回報人、缺失人員、分類、單號一起篩選；按「清除篩選」全部復原。

編輯視窗新增的欄位

點任一筆右邊的「編輯」，視窗下半部新增四個欄位，這些是缺失單列印要用的：

欄位填什麼印在缺失單哪裡
異常數量缺失造成的不良／損失數量不良數、異常數量（不良率自動算）
異常原因分析為什麼會發生（責任單位分析）（2）責任單位填寫 第一段
即時處理方式當下怎麼處理的（2）責任單位填寫 第二段
預防及修正方式以後怎麼避免再發生（2）責任單位填寫 第三段
不填會怎樣：缺失單一樣印得出來，只是這幾格會空白，要用手寫補。建議先在系統填好再印。

四、異常人員缺失單處理作業

位置：品保專區 → 異常人員缺失單處理作業（原名「異常人員統計」）。

查詢與篩選

  • 先選起始／結束日期，按「開始查詢」。
  • 明細表上方可用 部門 → 人員 兩層篩選：先選部門，人員清單只會出現該部門的人。

設定缺失處置

  • 單筆改：「缺失處置」欄的下拉選單直接改，選了就自動存檔。
  • 批量改：勾選多筆 → 上方出現「批量修改缺失處置」→ 選處置 → 按套用，會套用到所勾選每一筆的全部缺失人員。
批量修改請小心：會一次覆蓋所選紀錄中每一位缺失人員的處置，套用前會跳出確認視窗，請看清楚筆數再按確定。

看訂單明細

點「相關單號」欄的單號，會跳出訂單詳情視窗（品名、數量、交期等），不會離開目前畫面，看完關掉即可。

列印缺失單

  • 印一筆：該列最右邊的「列印」鈕（表格左右滑動時，這顆鈕會固定在右側不會跑掉）。
  • 印多筆：勾選多筆（表頭勾選框可全選）→ 上方「批量列印」。
  • 按下後直接下載 Excel 檔，檔名例如 缺失單_美編-怡妏_20260722.xlsx（含人名與日期）。

五、缺失單 Excel 怎麼看、怎麼印

  • 一位缺失人員一張單：一筆異常若有 2 位缺失人員，就會產生 2 個工作表（Excel 下方分頁）。
  • 批量列印時，所有單子都在同一個 Excel 檔裡，一個分頁一張。
  • 每張都已排好 A4 直式，開啟後直接 Ctrl+P 列印即可。

六、常見問題

填了 MOT 單號但沒有自動帶出資料？
① 確認單號沒打錯（MOT 開頭）；② 剛開的製令可能還沒同步進系統（每小時同步一次），可稍後再試或改填 SO 單號；③ 出現黃字提示時仍可正常送出，之後由品保在異常紀錄表補上品項資料。
缺失單印出來有些格子是空的？
系統只帶「已經填過的資料」。異常原因分析、即時處理方式、預防及修正方式要先在異常紀錄表編輯時填寫；沒填就會留白給你手寫。
不良率沒有顯示？
不良率＝異常數量 ÷ 訂單量。若「異常數量」沒填，或該單號在 ERP 查不到訂單量（例如手動輸入的非 ERP 單號），就會留白。
一筆異常有兩位缺失人員，會印成幾張？
兩張（Excel 兩個分頁），因為懲處是對個人的，一人一張才好簽核。
處置選單裡沒有「警告」這個選項？
目前下拉選項是口頭矯正／口頭警告／小過／大過。若要新增「警告」，可到品保專區 →「下拉選項管理」自行新增，列印時會自動勾對應的框。
編輯視窗太長，按不到儲存鈕？
新版已修正：視窗有高度上限，內容多時在視窗內捲動即可，儲存鈕一定按得到；視窗開啟時背景不會亂動。
批量修改缺失處置改錯了怎麼辦？
重新勾選那幾筆，再批量改成正確的處置即可；或在明細表用單筆下拉逐筆改回。
